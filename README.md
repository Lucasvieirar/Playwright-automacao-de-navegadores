# 🚀 Projeto de Automação com Playwright + Python

Este repositório contém exemplos práticos de automação de navegadores utilizando **Playwright com Python**, com foco em interações reais com páginas web e simulação de fluxos comuns em sistemas corporativos.

## 📌 Objetivo

Demonstrar, de forma clara e organizada, como utilizar o Playwright para:

* Automatizar ações em navegadores
* Interagir com elementos da página
* Extrair dados estruturados
* Simular fluxos de cadastro em sistemas

---

## 🧰 Tecnologias Utilizadas

* Python
* Playwright

---

## ⚙️ Funcionalidades Abordadas

### 🌐 Automação de Navegadores

Execução automatizada de ações como abertura de páginas, navegação entre telas e interação com elementos web.

### 📸 Captura de Tela

Registro visual das etapas da automação, útil para auditoria, testes e validação de processos.

### 🖱️ Localização e Interação com Elementos

* Localizando e clicando elementos utilizando a sintaxe padrão do Playwright
* Localizando e clicando elementos via seletores CSS
* Localizando botões e realizando ações utilizando XPath

### 🔍 Captura de Elementos

Identificação de múltiplos elementos na página para análise ou manipulação posterior.

### 🔄 Percorrendo Dados

Iteração sobre listas de elementos para coleta de informações relevantes.

### 📊 Extração de Dados e Atributos

Coleta de conteúdos textuais e atributos HTML de elementos da página.

---

## 🏢 Caso Prático

### Cadastro de Fornecedor no Sistema de Gestão de Estoque (SGE)

Simulação de um fluxo real de cadastro de fornecedor, incluindo:

* Preenchimento de campos
* Seleção de opções
* Envio de formulários
* Validação de sucesso da operação

---

## 📁 Estrutura do Projeto

O projeto está organizado de forma modular para facilitar o entendimento e reutilização dos exemplos:

* Scripts separados por funcionalidade
* Exemplos focados em diferentes estratégias de seleção de elementos
* Fluxos completos simulando cenários reais
